## Loading a file attachment

The launches dataset we'll show next lives in `launches.csv`, generated by `launches.csv.js`:

```js echo
const launches = FileAttachment("launches.csv").csv({typed: true});
```

A quick peek:

```js
Inputs.table(launches, {rows: 5})
```

## Inline HTML

You can mix arbitrary HTML into Markdown:

<aside style="border-left: 3px solid var(--theme-foreground-focus); padding: 0.5rem 1rem; margin: 1rem 0; background: var(--theme-background-alt);">
  This is an <code>&lt;aside&gt;</code> element rendered from inline HTML inside the Markdown page.
</aside>

<div class="note">This is a Framework "note" callout — just a div with class <code>note</code>.</div>

<div class="tip">Tips look like this.</div>

<div class="warning">Warnings are attention-grabbing.</div>

<div class="caution">Cautions are louder still.</div>
